# Smart-Home

## Objetivo 🎯
- Desarrollar una aplicación sobre una casa inteligente.

## Requisitos 🧩
- No se permite el uso de píxeles (px) para font-size, padding, margin o anchos de contenedor.
- Deben usar rem para tipografía y espaciados.
- Jerarquía de tres profundidades
- Deben crear un sistema de capas de abstracción para las variables.
- El dashboard debe usar grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)) en al menos una sección.
- El proyecto debe construirse bajo la filosofía Mobile-First. El CSS base debe ser la versión móvil, y las Media Queries deben usar exclusivamente min-width.

## Tecnologías 💻
- Figma para el diseño: https://www.figma.com/site/jRjpYJrrhpR1Cf83L9yNBh/proyectofinal?node-id=0-1&t=ayiFha1SG5GzEsfX-1 
- Html y css
- No se puede definir cada celda manualmente.
